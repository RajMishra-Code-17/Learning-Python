# 🎬 YouTube Manager

A simple Python CLI application to manage YouTube videos.

## 🚀 Features

- ➕ Add videos
- 📋 List videos
- ✏️ Update videos
- ❌ Delete videos
- 💾 Save data to a file
- ⚠️ Error handling

## 🛠️ Technologies

- Python
- File Handling
- Exception Handling

## ▶️ Run

```bash
python youtube_manager.py
```
